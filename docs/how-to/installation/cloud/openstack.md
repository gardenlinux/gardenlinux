# Install on OpenStack

This guide covers platform-specific installation and configuration tasks for deploying Garden Linux on OpenStack private clouds.

::: tip New to OpenStack Deployments?

For a complete step-by-step first-boot walkthrough, see the [First Boot on OpenStack](/tutorials/cloud/first-boot-openstack.md) tutorial. The tutorial covers uploading an image, launching an instance, configuring security groups, and connecting via SSH.

:::

## Post-Installation Steps

After installation, follow one or more of these post-installation steps to complete your OpenStack setup:

- [Creating a User Account using `cloud-init`](/how-to/installation/post-install.md#method-3-cloud-init-user-data-cloud-platforms)
