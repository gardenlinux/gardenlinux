# Install on GCP

This guide covers platform-specific installation and configuration tasks for deploying Garden Linux on Google Cloud Platform (GCP) Compute Engine.

::: tip New to GCP Deployments?

For a complete step-by-step first-boot walkthrough, see the [First Boot on GCP](/tutorials/cloud/first-boot-gcp.md) tutorial. The tutorial covers creating a custom image, launching a VM instance, configuring firewall rules, and connecting via SSH.

:::

## Post-Installation Steps

After installation, follow one or more of these post-installation steps to complete your GCP setup:

- [Creating a User Account using `cloud-init`](/how-to/installation/post-install.md#method-3-cloud-init-user-data-cloud-platforms)
