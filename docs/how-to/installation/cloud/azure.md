# Install on Azure

This guide covers platform-specific installation and configuration tasks for deploying Garden Linux on Microsoft Azure.

::: tip New to Azure Deployments?

For a complete step-by-step first-boot walkthrough, see the [First Boot on Azure](/tutorials/cloud/first-boot-azure.md) tutorial. The tutorial covers creating a virtual machine, configuring network security groups, and connecting via SSH.

:::

## Post-Installation Steps

After installation, follow one or more of these post-installation steps to complete your Azure setup:

- [Creating a User Account using `cloud-init`](/how-to/installation/post-install.md#method-3-cloud-init-user-data-cloud-platforms)
