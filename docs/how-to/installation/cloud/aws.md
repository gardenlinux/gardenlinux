# Install on AWS

This guide covers platform-specific installation and configuration tasks for deploying Garden Linux on Amazon Web Services (AWS) EC2.

::: tip New to AWS Deployments?

For a complete step-by-step first-boot walkthrough, see the [First Boot on AWS](/tutorials/cloud/first-boot-aws.md) tutorial. The tutorial covers selecting an AMI, launching an instance, configuring security groups, and connecting via SSH.

:::

## Post-Installation Steps

After installation, follow one or more of these post-installation steps to complete your AWS setup:

- [Creating a User Account using `cloud-init`](/how-to/installation/post-install.md#method-3-cloud-init-user-data-cloud-platforms)
