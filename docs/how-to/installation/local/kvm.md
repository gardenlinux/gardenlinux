# Install Locally Using KVM

This guide covers platform-specific installation and configuration tasks for deploying Garden Linux on KVM (Kernel-based Virtual Machine) using QEMU/libvirt.

::: tip New to KVM Deployments?

For a complete step-by-step first-boot walkthrough, see the [First Boot on KVM](/tutorials/local/first-boot-kvm.md) tutorial. The tutorial covers downloading the image, launching a VM with QEMU, configuring user creation via `fw_cfg`, and connecting via SSH.

:::

## Post-Installation Steps

After installation, follow one or more of these post-installation steps to complete your KVM setup:
