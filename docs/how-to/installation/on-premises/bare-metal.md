# Install Directly on Bare Metal

This guide covers platform-specific installation and configuration tasks for deploying Garden Linux directly on physical hardware (bare-metal servers).

::: tip New to Bare Metal Deployments?

For a complete step-by-step first-boot walkthrough, see the [First Boot on Bare Metal](/tutorials/on-premises/first-boot-bare-metal.md) tutorial. The tutorial covers writing the image to disk, partitioning, creating users via `chroot`, and first boot configuration.

:::

## Post-Installation Steps

After installation, follow one or more of these post-installation steps to complete your bare-metal setup:
