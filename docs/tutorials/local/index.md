# Local Tutorials

Run Garden Linux on your local workstation using virtualization. These tutorials require no cloud account and incur no cost.

## Prerequisites

- **Lima**: macOS or Linux with [Lima](https://lima-vm.io/) installed
- **KVM**: Linux host with KVM/libvirt installed
