# Cloud Tutorials

Deploy Garden Linux to public and private cloud environments. These tutorials require an active cloud account and incur usage charges.

## Prerequisites

- Active cloud provider account
- Configured CLI tools (AWS CLI, Azure CLI, gcloud, or OpenStack CLI)
- SSH client

## Cost Consideration

Cloud tutorials incur usage charges. Remember to clean up all resources after completing the tutorial to avoid ongoing costs.
