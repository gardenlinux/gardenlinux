# Garden Linux

<website-main>

<img align="left" width="80" height="80" src="https://raw.githubusercontent.com/gardenlinux/gardenlinux/main/logo/gardenlinux-logo-black-text.svg">
<a href="https://gardenlinux.io/">Garden Linux</a> is a
<a href="https://debian.org/">Debian GNU/Linux</a> derivate that aims to provide
small, auditable Linux images for most cloud providers (e.g. AWS, Azure, GCP
etc.) and bare-metal machines. Garden Linux is the best Linux for
<a href="https://gardener.cloud/">Gardener</a> nodes. Garden Linux provides
great possibilities for customizing that is made by a highly customizable
feature set to fit your needs. <br><br>

## !! Important !!

Starting with the major version 2017 the versioning scheme has been changed to a
semver compliant one comprised of a 3 numerical segment identifier
(major.minor.patch). The "old" 2017.0 is now 2017.0.0
